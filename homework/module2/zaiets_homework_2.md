## Завдання 1. Ієрархія каталогів Linux
1. Перехід в корінь: `cd /` та `ls` (виконано)
2. Перехід в /etc: `cd /etc` та `ls` (виконано)
3. Список користувачів: у середовищі Git Bash каталог `/home` відсутній, список користувачів Windows знаходиться за шляхом `/c/Users/`

## Завдання 2. Файли, каталоги та посилання
1. Створення каталогу: `mkdir ~/lab2`
2. Створення файлу всередині: `> base.txt`
3. Копіювання: `cp base.txt copy.txt`
4. Перейменування: `mv copy.txt renamed.txt`
5. Жорстке посилання: `ln base.txt hard_link.txt`
6. Символічне посилання: `ln -s base.txt sym_link.txt`
7. Пошук файлу: `find ~ -name "base.txt"`
*Результат пошуку:* `/c/Users/VLAD15/lab2/base.txt`

## Завдання 3. Права доступу
1. Перегляд прав: `ls -l base.txt`
2. Тільки на читання: `chmod 444 base.txt`
3. Право на запис власнику: `chmod u+w base.txt`
4. Значення umask: `0022`
5. Встановлення нового значення: `umask 022`

## Завдання 4. Користувачі (Емуляція для Linux)
1. Створення користувача: `sudo useradd -m trainee`
2. Додавання до sudo-групи: `sudo usermod -aG sudo trainee`
3. Перевірка: `grep trainee /etc/passwd`
*Приклад виводу:* `trainee:x:1001:1001::/home/trainee:/bin/bash`
